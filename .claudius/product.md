# Product Vision

`@nterprise-ai/dev` — single kitchen-sink dev-tooling package for the nterprise stack.

Lifted from the retired `@sharadkumar/web@0.10.x`. Provides:
- `nterprise` CLI — per-repo dev orchestration, multi-repo orchestration (`nterprise repos`), LaunchDaemon supervision, certificate management.
- Library exports — `themes` (OKLCH color engine), `website-builder` (React preview layer), tenant helpers, portless wrappers.

Published privately to GitHub Packages under the `nterprise-ai` org.
